for-click-handler

This happens to a lot of people new to JS/jQuery. Below, we want to click on A01 and fade B01, A02-->B02 and so on.

The code to add event listener is: 

$(document).ready(function(){

    for(i=0; i<10; i++){

        console.log('adding listener to ', '#a0' + i, ' fade target is ', '#b0' + i);
        $("#a0" + i).click(function (e) {

            e.preventDefault();

            console.log('#b0' + i + 'is fading') //what is the output if I clicked A01? What about A09?
            $('#b0' + i).fadeToggle('');
        });
    }
});
